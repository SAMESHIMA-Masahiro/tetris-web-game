# Tetris Web Game

HTML5 Canvasを使用したクラシックなテトリスゲームのWebアプリケーション

## 📋 プロジェクト概要

このプロジェクトは、ブラウザ上で動作するテトリスゲームを開発するためのものです。Vanilla JavaScript（ES6+）、HTML5 Canvas API、CSS3を使用して、フレームワークに依存しないシンプルで軽量な実装を目指しています。

## 🎮 主な機能（予定）

- **7種類の標準テトリミノ**（I, O, T, S, Z, J, L）
- **直感的なキーボード操作**
  - ←/→: 左右移動
  - ↑: 回転
  - ↓: ソフトドロップ
  - スペース: ハードドロップ
  - P: 一時停止
- **スコアリングシステム**
- **レベルシステム**（難易度が徐々に上昇）
- **ネクストブロック表示**
- **レスポンシブデザイン**

## 🛠 技術スタック

- **HTML5** - Canvas API を使用
- **CSS3** - モダンなスタイリング
- **JavaScript (ES6+)** - Vanilla JS、フレームワーク不使用
